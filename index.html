500 Dots

A visual experiment

Instructions:

  • Guide the other dots with the main white dot
  • Control the white dot with the following keys
  • "W" : Up
  • "A" : Left
  • "S" : Down
  • "D" : Right
  • To brake, tap "Spacebar"
  • Add more dots with "Q"
  • Pause the game with "P"
Blow away other dots with "Enter"!
